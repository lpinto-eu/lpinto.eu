About me

My name is Luís Pinto and I am a Software Engineer.

Currently I work and live in Aveiro, Portugal; I came here for university and soon fell in love with programming and with the city.

A dedicated Software engineer, highly committed to develop robust and maintainable solutions. Solid knowhow based on a strong academical background enhanced by a professional experience fulfilled with acknowledged good work - exciting challenges and lessons learned. Most of all, I'm always engaged to learn more on how to build even better software.

Background

After university I joined iUZ Technologies, where I continued the work with Java and webservices; there I had the opportunity to work in an international team.

I also worked in PT Inovação, building a new Secutiry system for buildings ingress control. It was based on a Javascript application and a server-side exposed through a REST API. We developed over an event driven and highly modular framework developed in-house. This framework is based on the principles of parallelism and asynchronism and was built based on the Akka toolkit and programmed in Scala.

I hope to find good professionals with whom I can learn; to be part in a team concerned with quality control leveraging automated checking tools; and most of all, contribute on building the tomorrow with tomorrow technologies.

OpenNCP - epSOS components.

Java + axis2 | SOAP + WS-* | XML | SAML

epSOS concentrates on developing a practical eHealth framework and ICT infrastructure that enables secure access to patient health information among different European eHealth systems. The OpenNCP is a conceptual collection of the common components specified by the epSOS project, with the aim of constructing a National Contact Point application.

Java + metro | SOAP + WS-*

Icarous - Parking Booking System.

Java + Hybernate + Struts2

ICAROUS offers the easiest way to book between all parking service options to ensure the availability and best price, It's a centralized reservation system providing access to parking options through aa large network of members. It shows a comparation of pricies, reviews and evaluations of all parkings. Allows the client to search, compare and make reservations.

RTS - Telematic Health Network.

Java + Hybernate + Struts2 + JSP | SOAP + WS-* | Postgres

The RTS platform aims to offer a regional eHhealth system in the region of Aveiro, allowing the partner institutions to exchange clinical data. Unlike point-to-point message or documents exchanging protocols, RTS builds on an architectural approach, exposing a coherent shared information model accessible through a Portal system.
